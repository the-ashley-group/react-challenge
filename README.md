# React Challenge

This project represents a standard configuration of a React project at The Ashley Group and can be used for interview purposes.

## Installation

```bash
# projet dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```

## Test

```bash
# all
$ npm run test

# react
$ npm run test:eslint

# eslint
$ npm run test:eslint

# stylelint
$ npm run test:eslint
```

## Lint

```bash
# prettier
$ npm run format
```
